# 🚀 Jhulon Kumar — Personal Portfolio

> A modern, animated personal portfolio website built with **React 19** + **Tailwind CSS v4** + **Framer Motion**.

🌐 **Live Demo:** [jhulon-kumar-portfolio.vercel.app](https://jhulon-kumar-portfolio.vercel.app/)

---

## 📌 About The Project

This is my personal developer portfolio website showcasing my skills, projects, and contact information. Built with the latest React 19 and Vite for blazing-fast performance, with smooth animations powered by Framer Motion and form validation using React Hook Form + Zod.

---

## ✨ Features

- 🎨 **Modern UI** — Clean and responsive design with Tailwind CSS v4
- 🌀 **Smooth Animations** — Page transitions and element animations via Framer Motion
- 📬 **Contact Form** — Validated form using React Hook Form + Zod schema validation
- 🧭 **Client-side Routing** — Multi-page navigation with React Router DOM v7
- ⚡ **Blazing Fast** — Powered by Vite v7 with SWC compiler
- 📱 **Fully Responsive** — Works seamlessly on mobile, tablet, and desktop
- 🔢 **Virtualized Lists** — Optimized rendering with TanStack Virtual

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm (comes with Node.js)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:5173
   ```

## 🌍 Deployment

This project is deployed on **Vercel**. To deploy your own version:

1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com) and import your repository
3. Set **Framework Preset** to `Vite`
4. Click **Deploy** — done! ✅
